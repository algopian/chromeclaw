# Keyboard Shortcuts

ChromeClaw supports keyboard shortcuts in the chat interface for common actions.

## Chat input

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift + Enter` | New line in message |
| `Escape` | Cancel current generation / close panels |
| `Up Arrow` | Edit last message (when input is empty) |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + B` | Toggle sidebar |

<Note>
Keyboard shortcuts may vary slightly between the side panel and full-page chat modes. Some shortcuts depend on browser-level bindings and may conflict with browser defaults.
</Note>
